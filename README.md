## Overview

This is project is aimed at demostrating the use of Elasticsearch for general purpose database and also for logging, together with kibana.

## Technologies Used
***
* ASP.NET Core
* Elasticsearch and Kibana
* Docker
* Serilog
* RabbitMQ
* Caching using Redis
